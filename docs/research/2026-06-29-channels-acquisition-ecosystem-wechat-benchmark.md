# 微信视频号获客生态全方位研究 × 视频号商业化矩阵对标

> 多 agent 并行研究（视频号商业化矩阵 / 视频号获客全景 / 竞品矩阵 / 获客内容打法）+ web 核验 + 源码映射。
> 与小红书、抖音获客研究同源方法，本次为视频号建对等获客体系（`channels` group + 9 阶段漏斗 + 12 技能 + 新平台 preset）。

## 一句话结论

**视频号是微信生态唯一「公域算法 + 社交关系链」双引擎入口，但它本身不闭环——价值在于把公域精准流量灌进企微/公众号/小店的私域池做长期复购。**
区别小红书（图文种草）/抖音（视频+直播+本地）的核心是 **微信公私域闭环（视频号↔公众号↔企微↔社群↔小程序↔朋友圈）+ 社交关系链裂变**。社交推荐贡献约 62% 初始流量，是冷启动命门。第三方 SaaS（微伴/句子互动/有赞/微盛/零一）做承接侧自动化，普遍弱在「内容与话术生产」——这正是 AICrew 的差异化卡位（内容上游，SaaS 承接下游，互补）。

## 二、视频号获客生态全景（2025-2026）

主链：`视频号内容/直播(公域获客) → 关注/加企微/进群(沉淀) → 公众号/小程序/小店(承接) → 社群+朋友圈(运营) → 转化复购 ↺ 反哺曝光`

**两套推荐机制**：好友推荐(社交链) + 平台算法推荐。社交推荐约 62% 初始流量——视频号重「内容能否撬动社交关系链」（区别抖音的「内容标签命中算法」）。
**6 大流量入口**：社交推荐流（点赞→约 20% 好友曝光、转发→约 50% 直接好友、收藏率>3% 出圈）/ 关注流 / 推荐流（3 秒播放率>50% 进池、完播率>45% 二次推流）/ 搜一搜 / 直播流（预约撬动+留资组件需在线≥10min）/ 附近流。
**2025-2026 主旋律**：严管硬导流 + 鼓励生态内合规跳转 + 内容价值优先。

### 合规红线（视频号差异化护城河）

- 已禁止主页/评论/私信硬导微信、二维码、诱导分享利诱关注。
- 合规路径：「一体号」（视频号绑公众号→关注自动回复推私域）、公众号菜单、直播留资组件、短视频挂企微（内测）。
- 这是 `channels_compliance_guard_v1`（导流合规红线校验改写）的存在理由。
