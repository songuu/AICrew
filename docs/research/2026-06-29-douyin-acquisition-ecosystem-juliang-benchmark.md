# 抖音获客生态全方位研究 × 巨量引擎/抖音电商产品矩阵对标

> 多 agent 并行研究（巨量/抖音商业化矩阵 / 抖音获客全景 / 竞品矩阵 / 获客内容打法）+ web 核验 + 源码映射。
> 与小红书获客研究同源方法，本次为抖音建对等获客体系（`douyin` group + 9 阶段漏斗 + 14 技能）。

## 一句话结论

**抖音商业化的执行层（千川投放/账号自动化/平台配置）AICrew 不碰；可吃下的是「内容/话术/脚本/SOP/策略产物上游」。**
抖音相对小红书多三大获客支柱：**直播获客、本地到店、千川/DOU+ 投流**——这是抖音漏斗区别于小红书图文种草主线的核心。AICrew 是这些抖音获客 SaaS（快鲸/矩阵 SaaS）的「内容引擎上游」，承接「生成什么」，不承接「自动执行」。

## 二、抖音获客生态全景（2025-2026）

主链：`内容触达(推荐/搜索/同城/直播/货架/粉丝) → 互动 → 线索捕捉(评论/私信/表单) → 私域承接(企业号/粉丝群/企微) → 转化复购(直播/商城/团购)`

**2025 范式变化**：① 算法去标签化重「人」，硬广完播率同比暴跌约 37%；② 搜索成第二增长极（日活搜索 7 亿+，搜索转化 +35%）；③ 本地生活高速扩张（生活服务 GTV 同比 +60%）；④ 千川「全域推广-净成交出价」升级，AIGC 批量产素材成标配——「素材定生死」；⑤ **私域引流强监管**：AI 识别「直接留微/诱导跳转」，私信封 7–30 天，直播间挂企微二维码可永久封禁。

**6 大流量入口核心指标**：推荐（前 3 秒留存权重~40% > 完播率~40% 门槛；互动权重 转发>评论>点赞>收藏；行为价值 关注>收藏>评论>点赞）/ 搜索（关键词排名/搜后转化）/ 同城（POI 点击/到店/核销）/ 直播（停留/互动/UV价值/留资率）/ 货架（商品卡 CTR/复购）/ 粉丝（转粉率/私信回复）。

**直播话术 SOP**（契合度最高）：开场暖场（开播至上品≤8 分钟）→ 留人 → 憋单（库存倒数/限量）→ 逼单（3·2·1 上链接）→ 留资承接 → 结尾。

## 三、竞品矩阵对标

已核验：蝉妈妈、飞瓜、抖查查、新抖、灰豚、巨量算数（数据）；快鲸/各类矩阵 SaaS（获客自动化）；句子互动/微伴 SCRM。
**AICrew 已覆盖产物侧**：带货素材、达人 brief、转化话术、内容生成。
**可建缺口**（内容/策略产物）：账号定位、搜索SEO、引流短视频脚本、爆款改写、矩阵起号、直播脚本、探店/团购、千川素材、人群反漏斗、评论截流、线索留资、企业号私域、数据复盘。
**设计性出域**：实时数据监测、达人数据库、自动私信/批量发布、排名查询、CRM/SCRM。

## 四、落地决策（本次 sprint）

新增 **14 个抖音获客技能** + **9 阶段全新漏斗**（`douyinPromotionStages`）+ **3 个抖音钩子框架**（完播钩/直播憋单钩/直播逼单钩）。
架构：镜像小红书 rednote 体系，并把推荐评分内核抽成通用 `recommendPromotionSkills`（rednote/douyin 共用，DRY）。详见 `docs/plans/2026-06-29-douyin-acquisition-expansion.md`。

漏斗：`定位诊断 → 搜索卡位 → 内容引流 → 直播获客 → 本地到店 → 投流放大 → 线索留资 → 私域承接 → 数据复盘`
