# 小红书获客生态全方位研究 × 小鸡AI 产品矩阵对标

> 基于多 agent 并行研究（小鸡AI 矩阵 / 获客生态全景 / 竞品矩阵 / 获客内容打法）+ web 核验 + 仓库源码逐项映射。
> 结论代表 2026-06-29 仓库状态（`lib/domain.js` 小红书技能 19 → 31，漏斗 7 → 9 阶段）。

## 一句话结论

**小鸡AI 的护城河是「自动化执行」（采集 / 养号 / 自动私信 / 批量发布）——恰是 AICrew 不碰、且小红书 2025 起强制合规绞杀的部分。**
AICrew 能吃下的是**内容与策略产物层**：把小鸡AI 的功能「反编译」成一组合规、可静态站交付、复购属性强的「话术 / SOP / 模板 / 选题」生成技能。半映射（监控/打标 AICrew 做不了执行）降维成「判定标准 + 对应话术产物」，覆盖客户 80% 真实需求（他们要的是「说什么」，执行可手动或交官方组件）。

## 映射图例

- ✅ 可映射（内容/话术/模板/策略产物形态交付） | ⚠️ 半映射（只能产出内容产物，执行/自动化做不了） | ❌ 不可映射（纯爬虫/自动私信/账号自动化）
- [核验] 多源网络确认 | [推断] 网络不足，领域知识补全

## 二、小红书获客生态全景（2025-2026）

主链：`内容触达(发现页/搜索/同城) → 互动 → 线索捕捉(评论钩子/私信) → 私域承接(私信通→企微/微信) → 转化复购 ↺ 反哺公域`

- **流量机制**：CES 赛马 `赞×1+藏×1+评×4+转×4+关注×8`，黄金 2 小时定生死；搜索占站内 60%+ 是转化主战场；同城 LBS 做到店；群聊=站内半私域；私信=唯一合规留资闸门。
- **主动截流**：① 评论区截流（去高赞/竞品/KOL 笔记发「神评」，靠内容力引流到主页，评论区禁联系方式）；② 关键词卡位截流（下拉/相关搜索扩词 → 矩阵卡位首屏）。
- **引流钩子**：诱饵要具体可感（「30 页选品清单」非「资料包」）；钩子=价值前置 + 留白 + 唯一低门槛行动指令（扣 1/私信关键词）。
- **官方工具定位**：薯条加热单篇 / 聚光精准投流（也是私信通前置）/ 蒲公英找达人 / 专业号拿资格 / 群聊养私域 / 店铺做成交 / 千帆-私信通收线索。

### 防违规导流红线（2025 新规，硬性）

- **绝对禁止**：笔记/评论/主页/私信自动回复出现微信号、手机号、二维码、外链、谐音变体（威/V/薇）。
- **唯一合规路径**：私信通-私信获客工具 →（服务留资卡 / 企微名片 / 个微名片 / 快捷菜单名片），需专业号 + 开通聚光。
- 这条红线是 AICrew **「合规违禁词体检改写」横切技能**的存在理由，也是相对小鸡AI 这类灰产工具的差异化护城河。

## 三、竞品矩阵对标（行业功能全集）

已核验：新红(newrank)、千瓜、灰豚、蝉小红、灵犀(官方)、矩阵通、蒲公英/聚光/薯条/千帆、句子互动 SCRM、来鼓/美洽 AI 获客、零克查词、Reditor、易媒。

**关键发现**：现有 19 技能基本未用到 **persona / trend** 两个闲置 agent 节点，而它们恰对位行业最高频的「人群画像/人设定位」「热点趋势/数据选题」缺口——补这两类技能零新增基建。

**AICrew 已对齐甚至超越竞品的产物侧**：内容创作、达人 brief、投放素材、转化话术、复盘。
**真正可建缺口**（均为内容/策略产物）：合规质检、人群/人设、卖点诊断、数据选题、私域路径、舆情话术、爆文改写、矩阵起号、引流钩子、截流话术。
**设计性出域**（不追）：实时数据监测、达人数据库、自动私信机器人、多账号批量发布、客资 CRM。

## 四、落地决策（本次 sprint 实现）

新增 **12 个获客类小红书技能** + **2 个漏斗阶段**（`lead_capture` 线索捕捉、`private_domain` 私域承接）+ **3 个获客钩子框架**（诱饵留白钩 / 截流神评钩 / 痛点筛选钩）。详见 `docs/plans/2026-06-29-rednote-acquisition-expansion.md`。

漏斗演进（7 → 9 阶段）：
`诊断定位 → 搜索策略 → 内容生产 → 达人种草 → 投放放大 → 线索捕捉(新) → 转化承接 → 私域承接(新) → 复盘优化`
